# Login Form UI

A simple and responsive Login Form UI built using HTML and CSS.

## Features

- Clean and modern design
- Responsive layout
- Username and password fields
- Login button
- Easy to customize

## Technologies Used

- HTML5
- CSS3

## Project Structure

```
LoginFormUI/
│── index.html
│── style.css
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your web browser.
